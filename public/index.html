📡 Webhook Setup

To connect GitHub events to this dashboard:

  1. Go to your repo Settings → Webhooks → Add webhook
  2. Payload URL: https://YOUR-DEPLOYMENT.vercel.app/api/webhook
  3. Content type: application/json
  4. Events: Select "push", "pull_request", "issues", "workflow_run"
  5. Add secret (optional): Set WEBHOOK_SECRET in Vercel env vars

Add GITHUB_TOKEN to enable autonomous issue creation.
